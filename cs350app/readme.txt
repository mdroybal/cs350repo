Youtube Audio Video DL Readme

=============================

The purpose of this application is to have the ability to download video or audio from a youtube url.
The application utilizes the the youtube-dl package to do so. The application utilzes a keyboard input to perform desired action. The user should have the ability to download high and low quality audio and video with the application. The users of this application should be any one who desires to dowload audio or video files to be stored to a directory. 

The requirements of the this application include Python 2.7.12 to be utized along with the youtube-dl==2016.10.12 library and sys library. Another system requirement may be the -y libav-tools- to convert the audio format when an avprobe or ffprobe is not available.
